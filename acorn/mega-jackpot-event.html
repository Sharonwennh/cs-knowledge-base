🎮 How to Join

Players are not enrolled by default. They must manually opt in.

Method 1: Via Slot Machine

  • Enter any slot machine
  • See the Mega Jackpot pool carousel at the upper left corner
  • If already participating, the upper right corner displays "Joined"
  • Click the carousel icon → opens event page → click "Join Jackpot"

Method 2: Via Lobby

  • Click the Mega Jackpot icon at the bottom right of the Lobby page
  • Click "Join Jackpot" to participate or "Opt Out" to leave
✅ Eligibility: Only SC mode spins are eligible. All slot machines qualify.

📊 Event Rules

1. Four Jackpot Pools

Each SC spin gives a chance to win the current prize of any pool. All players share the same jackpot amounts and winning odds.

  • Each jackpot has its own countdown timer
  • If a player wins, the countdown resets immediately
  • If no one wins before countdown ends, the next spin guarantees a win for that pool

🏆 Winning & Prize Distribution

  • Jackpot is randomly triggered during the game spin
  • Winning animation displays — valid for current spin only, cannot be claimed repeatedly
  • Prize is automatically added to player's Unplayed SC balance
  • Winning amount must comply with same redemption restriction rules as other SC prizes
  • Only Played SC can be redeemed; Unplayed SC cannot be redeemed

Q2: Why is X SC deducted extra for each spin after I join?

Script:

Dear Player,

The additional X SC deducted per SC spin is your contribution to the Mega Jackpots event. This amount is distributed to the four jackpot pools (Mini, Hourly, Daily, Mega) to fund the prizes.

You can click "Opt Out" in the event interface at any time to stop the deduction and exit the event.
*The amount of SC deduction please refer to the player's description or the screenshots they have provided.

Q3: I won a jackpot! Where is my prize, and can I redeem it immediately?

Script:

Congratulations on winning! 🎉

Your prize has been automatically added to your Unplayed SC balance.

Please note that only Played SC can be redeemed; Unplayed SC cannot be redeemed directly. The winning amount follows the same redemption restriction rules as other SC prizes.

Q4: Why don't I see the Mega Jackpot event entry?

Q5: Is the "Guaranteed Drop Countdown" real? Will the jackpot definitely be won?

Script:

Dear Player,

Yes! Each jackpot pool has a guaranteed drop countdown. If no one wins the jackpot before the countdown ends, any player who placed the next SC spin will definitely win that jackpot.

If the jackpot is won before the countdown ends, the timer will reset immediately. All participating players are treated with integrity!
