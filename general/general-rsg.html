一、RSG 限制设置（通用）

1.1 限制类型

类型说明项目组差异
Purchase Limit购限（充值金额上限）各项目组可能不同
SC Limit游玩SC上限（每日可消耗的SC上限）各项目组可能不同
Playtime Limit每日游玩时间上限各项目组可能不同

1.2 设置成功确认

  • 玩家直接表达RSG相关内容 → 直接走RSG流程
  • 设置成功后需查看截图确认
  • 反馈当前设置的上限值
  • 若未成功设置任何limit → 重新发送链接

二、Take a Break（通用）

2.1 流程

  1. 即刻生效
  2. 玩家在RSG通道查看是否成功（显示起始和终止日期）
  3. 成功设置后提交工单请求二线unfreeze账号
  4. 二线核验成功后unfreeze
  5. 一线回复玩家有效期

2.2 数字ID玩家

  • 如果玩家为数字ID，需二线查看是否成功设置

三、Self Exclusion（通用）

3.1 流程

  1. 玩家设置后先在RSG通道查看是否成功
  2. 成功设置后提交工单请求二线Ensure Self-exclusion
  3. 二线核验是否成功设置并同时unfreeze账号
  4. 一线回复有效期

3.2 玩家直接来信要求Self Exclusion

  • 一线先冻结账号
  • 再发送RSG链接或向二线申请RSG link

四、删号/Freeze流程（通用）

4.1 玩家要求Close Account

  1. 先RSG → 再Freeze

4.2 玩家要求Delete Account

  1. 告知不可逆，提供RSG选项
  2. 如果玩家坚持删号：
    • 提交工单让二线删除
    • 一线操作Ban list + 封号note【玩家坚持删号】

4.3 可解决问题优先解决

  • 可解决问题 → 优先解决
  • 不可解决 → 提供冻结选项
  • 不接受 → 提交工单

五、Function Not Available Yet（通用）

弹出该提示或玩家为纯数字ID时，说明产品还不能直接生成RSG。

5.1 处理流程

  1. 提工单向二线索要rsg link
  2. 给玩家提供Freeze账号选项
  3. 如果需要Freeze → 先Freeze等RSG link后再发送
  4. 如果不需要Freeze → 等待二线更新工单rsg link再发送

六、RSG链接通用话术

Sorry for the delayed response... Please kindly open the below link... set the purchase limit... within 24hrs otherwise the link will expire.

Link address: xxx

Please let us know if you need further assistance. Have a great day.

七、术语修正（通用）

  • "自我封禁" 在中文客服语境中等同于玩家自己删除/封禁了自己的账户
  • 英文应译为 "account deletion" 或 "self-deletion"
  • 而非 "Self Exclusion"（后者是负责任游戏暂停工具）
