📝 How to Redeem / 如何提现

General Redemption Guide / 通用提现指南

Please click REDEEM to check the rules.

  • 1 sweepstakes coin = 1 USD
  • Sweepstakes coins must be used to play at least once to be eligible for redemption.
  • All the SC you win can be redeemed.
  • GC cannot be redeemed or converted to SC.
  • Normally a redemption takes less than 3 days to process.
  • Each redemption request is reviewed manually to ensure account safety.

请点击 REDEEM（提现） 查看规则。

  • 1 SC = 1 USD
  • SC 必须至少玩过 1 次才能提现。
  • 所有赢得的 SC 均可提现。
  • GC 不能提现，也不能转换为 SC。
  • 通常提现处理时间不超过 3 天。
  • 每笔提现请求均人工审核，以确保账户安全。
Dear Player, Thanks for your patience. Please click REDEEM to check the rules. 1 sweepstakes coin = 1 USD, sweepstakes coins must be used to play at least once to be eligible for redemption. All the SC you win can be redeemed. GC cannot be redeemed or converted to SC. Normally a redemption takes less than 3 days to process, each redemption request is reviewed manually by our agents to ensure the safety of players' accounts. So it may need a little bit more time. Thanks for your patience. If you have more questions please kindly send us screenshots of your issue so we can help you check further. Have a great day.

  3. Provide: clear selfie + photos of front and back of credit card used for previous purchases

如玩家验证信息不一致，提现时会出现 Payment Check 弹窗。玩家可关闭弹窗正常提交提现，但弹窗会在后续提现时持续出现。建议玩家完成验证：点击 Confirm → 跳转 Sumsub 平台 → 提供清晰自拍 + 之前购买使用的信用卡正反面照片。

📐 Table Format Rules / 表格格式规范

Redemption Table Format Requirements / 提现表格制作要求
STATUS DISPLAY RULE — CRITICAL UPDATE:
In all player-facing tables, status must be displayed as TEXT ONLY. Backend numeric codes are for internal use only and must NEVER be shown to players.

Examples of WRONG display: "3redeemed", "116 Need to Collect", "4Failed", "5 Canceled"
Correct display: "Redeemed", "Need to Collect", "Failed", "Canceled"

General Table Format / 通用表格格式

  • Header: Dark blue (#2F5597 or #4472C4), white text
  • Data rows: White / Light blue (#D9E2F3) alternating
  • Column names in English: Player ID, Created Time(UTC), Status, etc.
  • Time columns must be labeled UTC
  • Amount columns: Use $ symbol for player-facing amounts
  • Title format: Redemption Record(s) — [PlayerID]

Required Columns for Redemption Tables / 提现表格必含列

  • Player ID / Nickname
  • Account Email
  • Status (text only, no numbers)
  • Redemption Amount (with $)
  • Created Time (UTC)
  • Redemption Method
